質問 3

Day6 で作るべき API サービスの要件を定義してください

回答 3

FastAPI の学習計画において、Day6 で作るべき API サービスの要件を以下に定義します。

Day 6: API サービスの開発

要件定義

  • API サービスは、ユーザー認証・認可が必要な機能を提供する
  • ユーザーは、サインアップ、サインイン、ログアウトができる
  • ユーザーは、アカウント情報の閲覧・更新ができる
  • ユーザーは、投稿の作成・閲覧・更新・削除ができる
  • 投稿には、タイトル、本文、投稿者、投稿日時が含まれる
  • 投稿には、複数の画像が添付できる
  • 投稿には、いいねがつけられる
  • ユーザーは、他のユーザーの投稿にいいねがつけられる
  • ユーザーは、自分がつけたいいねの一覧を閲覧できる
  • ユーザーは、他のユーザーのプロフィールを閲覧できる

上記の要件を満たす API サービスを開発することで、FastAPI の基本的な機能や機能拡張を学ぶことができます。また、フロントエンドの React との組み合わせで、完全な Web アプリケーションの開発に挑戦することができます。

ディレクトリ構成とDockerfile

元のディレクトリは以下のものから始めた。

.
├── .github
│   ├── actions
│   │   └── build
│   │       ├── Dockerfile
│   └── workflows
│       └── main.yml
├── .gitignore

自分用にプロプリプロセッサを作ったため、それをgithubactionsで使用することにした。そのためにはどこかに置いてcargo installすれば良いことはわかっていたので、とりあえずルートにおいてDockerfileでのinstallを試みた。

.
├── .github
│   ├── actions
│   │   └── build
│   │       ├── Dockerfile
│   └── workflows
│       └── main.yml
├── .gitignore
├── mdbook-hiromatter #自作のプリプロセッサ

Dockerfileを使ってbuildを行うが、どうやってもcargo installができない。ディレクトリが参照できず、エラーを吐いて終わってしまう。

dockerのbuildを行う際は/.github/actions/buildの中を参照しているのだから、その中に入れることにした。

├── .github
│   ├── actions
│   │   └── build
│   │       ├── Dockerfile
│   │       ├── mdbook-hiromatter
│   └── workflows
│       └── main.yml
├── .gitignore

これでうまくいった。buildディレクトリの外側はbuild時に読み込めないので、その中に入れるしかないような気がするんだけどどうなのか。

部活が終わった今思うこと

先日４年生を送る会なるものが行われ、僕の４年間の部活生活は完全に終わった。前に書いたときはシーズン中でしっかり振り返りきれてないような気もするから、一応今の自分の考えを少し残しておこう。

競技自体も好きだし周りの人も当たり前に大学でラグビーを続けてたから、大学でラグビーで関わらないという選択肢はなかった。ただ、併願校的にも第一志望受からなかったら選手を続けるのは無理そうと思ってたから（実際にスタッフ希望のメールも送っていた。今思えば続ける勇気がなかっただけなんだけど。）、受かった時にこれでラグビーできるなと嬉しかったのは今でも覚えてる。合格発表の日にはラグビー部の人を見つけて話しかけに行ったし、入学式の前に入部届を出して入部した。ラグビーをすることが当たり前だったし入部することに何の悩みもなかったから、今振り返っても自分がした選択に対して何も思うことはない。今１年生になっても絶対ラグビー部を選ぶし、というか選ばざるを得ないというか。

この４年間、ラグビー出来るってだけで僕は満足だった。きつかったり辛いことは山ほどあったけど、少しやる気がなくなってしまったことはあったけど、一度もラグビーを辞めたいとは思うことはなかった。部活を続ける理由については４年間でよく議論されていて、勝つためとか人が好きだからとかあげようと思えば幾つでもあげられるけど、結局はラグビーが好きっていうのが根本にあったし一番大きな理由だった。やればやるほどラグビーの奥深さに触れることができたし、知ろうとすればするほど新たな顔を見ることができた。10 年以上やってきたけど新たな発見ばかりで飽きることがなかった。ラグビーは最高なスポーツだと確信している。

やっぱり僕はラグビーが好きだし、僕にはラグビーしかないんだな。後輩とかコーチで教えた子供たちとか、他の人たちがラグビーの魅力を知ってラグビーを好きになっていく姿を見ると、本当に嬉しいし幸せに思う。これから先ラグビーに出会う人たちに何を一番伝えたいかって考えると、もちろん部活とかで真剣にやるんだったら最後まで続けてほしいっていうのはあるんだけど、とにかくみんなにはラグビーを好きになって欲しい。好きになるまでは時間がかかるかもしれない。あんなにも辛くてきついスポーツはないのだから。だけどいつか好きになってくれたら、少しでも楽しいと思ってくれたら、ラグビーの先輩としてこれ以上幸せなことはない。

ラグビーとは

「ラグビーは少年をいちはやく男にして、男に永遠の少年の魂を抱かせる」 by Jean-Pierre Rives

ラグビーの起源

1800 年前半、英ラグビー校の生徒ウェブ・エリスがフットボールの試合中にボールを持って走り出しラグビーが始まったとされている。
イギリスフットボール協会 FA (Football Association) 式であるサッカーと、ラグビー協会 RFU (Rugby Football Union)式であるラグビーの二つが存在している。前者をア式蹴球と呼び、後者をラ式蹴球と呼ぶのはその名残。
イギリス起源のスポーツであり、イギリスの植民地支配と合わせて世界各地に広がっていった。

日本におけるラグビー

1899 年、慶應義塾にラグビー部が創立される。以来 125 年間、様々な人に愛され支えられながら現在まで発展してきた。

1980 年代、日本におけるラグビー人気は最高潮を迎える。1984 年には高校ラグビーを題材としたスクールウォーズが TBS でドラマ化され爆発的な人気となった。この時期は新日鐵釜石が日本選手権 7 連覇を達成する時期と重なり、また、大学ラグビーが熱狂的な人気を博した時代であった。
1982 年の早明戦には収容人数を超える 66999 人の観客が国立競技場に集まり、入場できなかった観客たちはフェンスの外から試合を眺めるのであった。1987 年の早明戦は「雪の早明戦」として知られ、雪が降り積もる中熱戦を繰り広げる。現在でもこの試合は語種であり、白い雪と暑く激しいラグビーのコントラストが今でも日本ラグビー史に輝いている。

2000 年代は日本ラグビー冬の時代であった。W 杯では勝てず、2003 年に始まった社会人ラグビーリーグ「トップリーグ」も観客動員数が伸び悩み、25000 人収容の秩父宮ラグビー場に 1000 人も人が集まらない試合も珍しくはなかった。僕がラグビーを始めたのは 2010 年だったが、2011 年のラグビーワールドカップは 0 勝 3 敗 1 分でその１つの引き分けに皆で大喜びしたのを覚えている。

2012 年、日本ラグビーに救世主が現れる。サントリーサンゴリアスのヘッドコーチを務めていたエディ・ジョーンズが日本代表の監督に就任した。「JAPAN WAY」をスローガンに掲げ、日本にしかできない新しいラグビーを追求していく。
2013 年にはヨーロッパの強豪ウェールズ相手に勝利、ヨーロッパ遠征で初めて敵地で勝利するなど着実に成長をとげる。

#RSAvJPN #RugbyWorldCup You couldn't write this... by J.K. Rowling

そして迎えた 2015 年、イングランドでワールドカップが開幕する。9/19、イギリスは南部ブライトン。ワールドカップ二度の優勝を誇る南アフリカと通算成績 1 勝 21 敗 2 分の日本がぶつかる。全世界の人間が南アフリカの勝利を予想、オッズは南アフリカが 1 倍、日本が 34 倍であった。
試合は日本が先制。五郎丸歩のペナルティーキックで 3 点を得る。すぐにモールから南アフリカにトライを返されるが、日本も同じくモールで 7 点を追加する。その後南アフリカに 5 点を追加され、10-12 で前半を折り返す。
そして後半、日本は五郎丸の PG を主体に得点を積み重ね、南アフリカにトライを取られながらも接戦に持ち込む。後半 28 分ラインアウトからサインプレー「府中 12 内」が決まりついに同点に追いつく。しかし引き下がらないのが南アフリカ。後半 32 分にゴール前のペナルティでショットを選択し 3 点を追加する。（ここで PG を選択してしまった時点で南アフリカは日本相手に怯み負けているということではあるのだが）
残り時間もあとわずかの中、日本代表は相手のプレッシャーを受けながらも敵陣ゴール前まで攻め続ける。そして試合終了間際、ゴール前でペナルティを得る。ここでショットを決めれば 3 点、同点で試合を終えることができる。弱小国が巨人相手に同点に持ち込むだけでも世界は賞賛しただろう。しかしキャプテンのマイケルリーチは違った。3 点で同点を狙うのではなく、とにかくトライ、5 点を取り勝利だけを目指した。40 分を告げるホーンはすでに鳴った。残りラストワンプレー。ボールがアマナキレレイマフィに渡る。ハンドオフで一人かわし、ボールは完全に前が空いたカーンヘスケスのもとへ。ゴールへ飛び込み 5 ポインターとなる。ラストワンプレーで奇跡の逆転。34-32 で日本が勝利した。

その後は 3 試合行われ、弱小国ゆえの過密日程も影響し 2 勝 1 敗。3 国が 3 勝 1 敗で並ぶ結果となったがボーナスポイント等のためプールステージ 3 位。惜しくも決勝トーナメントへ進むことはできなかった。

4 年に一度じゃない。一生に一度だ。 -ONCE IN A LIFETIME-

2019 年、日本でワールドカップが開催される。前回大会達成できなかったベスト 8 を目標に選手たちは 4 年間ハードなトレーニングを積んできた。プールステージで当時世界ランク 1 位のアイルランドと静岡エコパスタジアムで激突する。前回大会での日本の活躍を見ている緑の戦士たちは一つの油断もなく日本にぶつかってくる。序盤に 2 トライを奪われる展開となったが、田村優が PG を落ち着いて決め続け 9-12 で前半を折り返す。
後半、福岡堅樹が値千金のトライを決め 16-12。その後自陣に貼り付けの時間を過ごすが福岡のインターセプトもあり敵陣に攻め込みそのままノーサイド。19-12 で世界トップのチームから勝利を奪う。
前回大会苦渋を舐めさせられたスコットランドからも勝利を収め、プールステージ全勝、4 勝 0 敗で決勝トーナメントへ駒を進める。

決勝トーナメント初戦の相手は前回大会金星をあげた南アフリカ。南アフリカの壁は高すぎた。1 トライも奪えず 3-26 で完敗。世界との差を感じる結果となった。

ルール

Apart from the playing of the game and its ancillary support, rugby embraces a number of social and emotional concepts such as courage, loyalty, sportsmanship, discipline and teamwork. Laws of the game

基本的なルール

まずラグビーは 15-15 で行われる楕円型のボールを使ったスポーツです。アタック側はラン・パス・キックでゴールを目指し、ディフェンス側はタックルとその後のブレイクダウンの攻防でボール奪取を狙います。その一連の攻防の中で反則があるとスクラム・フリーキック・ペナルティキックのどれかでゲームが再開します。

とりあえず押さえておくべきルールは以下の 2 つ。

  • ボールを前に投げてはいけない
    • それを守らないとスローフォワードという反則
  • ボールを前に落としてはいけない
    • それを守らないとノックオンという反則

あとは危険なプレーは基本的に反則です。頭にタックルいくとか、空中にいる相手にタックルいくとか。
難しい反則も非常に多いので一緒に一度見ましょう。

ポジション

まずは大きくわけて 2 つ。スクラムを組む FW（フォワード）とスクラムを組まない BK（バックス）。FW は体が大きくて BK はキレのある体なイメージで大丈夫。背番号 1~8 が FW で 9~15 が BK です。

  • FW
    • PR: プロップ 背番号 1,3
      • スクラムを最前線で組みます（フロントロー）
      • 後述の HO を挟んで左側が 1 番の左プロップ、右側が 3 番の右プロップ
    • HO: フッカー 背番号 2
      • スクラムを最前線で組みます（フロントロー）
      • ラインアウトでボールを投げ入れる場合が多い
    • LO: ロック 背番号 4,5
      • スクラムを 2 列目で組みます（セカンドロー）
      • 長身が多く、ラインアウトでジャンパーを務めることが多い
    • FL: フランカー 背番号 6,7
      • スクラムを 3 列目で組みます（サードロー、バックロー）
      • スクラムで自由な場所にいるため、とにかくタックルする機会が多い
    • No8: ナンバーエイト 背番号 8
      • スクラムを 3 列目で組みます（サードロー、バックロー）
      • とにかく花形のポジション
      • アタックもディフェンスもなんでもできる人がやるポジション
  • BK
    • SH: スクラムハーフ 背番号 9
      • スクラムにボールを投入し、最初にパスを投げる
      • オープンなプレー中も最初にパスを投げる
    • SO: スタンドオフ、フライハーフ 背番号 10

ワンランク上を目指して

まずラグビーの基本的な大原則として、「ボールの争奪と継続」という概念がある。ボールを保持していない側はボールを奪い、保持している側はそのボールを継続してトライを狙う。その両チームのバトルこそがラグビーの醍醐味であり、ラグビーをラグビーたらしめる要素なのだ。なので、その争奪と継続を途切れさせるプレーには反則が科される。

e.g. ノットリリースザボール
AT 側が倒されてラックが成立した際にボールを離せないと AT 側の反則となる。それは DF が争奪しようとした結果 AT が継続できなくなっているためである。

e.g. ノットロールアウェイ
AT 側が倒されてラックが成立した際に、DF 側が邪魔になりボールを出せないと DF 側の反則となる。それは AT 側が継続しようとしているのにも関わらず、DF 側が正しくない争奪をすることで継続不可になっているからである。

「争奪と継続」を意識してラグビーを見ることでわかりにくいラグビーのルールがクリアになるだろうと私は考えている。
